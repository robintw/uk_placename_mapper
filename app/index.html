UK Placename Mapper

This tool lets you visualise UK place names that match certain search terms on a map. Search terms can be plain text, or you can use regular expressions. For example:

  • sea - any place whose name contains sea
  • ^newton$ - any place whose name is exactly newton
  • burgh$ - any place whose name ends with burgh
  • ^south - any place whose name starts with south
